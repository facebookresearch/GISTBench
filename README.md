The data is released under [CC-BY-NC 4.0](LICENSE) and is intended for benchmarking purposes only.

The `object_text` portion of the data are outputs of Llama 3.2 and subject to the [Llama 3.2 license](https://github.com/meta-llama/llama-models/blob/main/models/llama3_2/LICENSE). If you use this portion of the data to create, train, fine tune, or otherwise improve an AI model, which is distributed or made available, you shall also include "Llama" at the beginning of any such AI model name.

## Data Description

| Field | Description |
|---|---|
| `interaction_type` | Actual interaction type of the user x item from the aggregate content pool, aggregated to 3 types. |
| `user_id` | Anonymized user identifier (1-N). |
| `object_id` | Anonymized object identifier (1-N). |
| `interaction_time` | Anonymized interaction time. |
| `object_text` | Generated by a VLM to summarize the video frame by frame then using Llama 3.2 70B to create hashtags from the video summary. |
